Unified Messaging

최종 업데이트: 2026.06.19  |  번역: MCE 구축 가이드 팀

개요

Unified Messaging은 Marketing Cloud와 Service Cloud Digital Engagement를 이메일·SMS·WhatsApp 등 서드파티 메시징 앱에 연결합니다. 필요에 따라 모든 채널 유형을 구현하거나 하나만 선택할 수 있습니다.

💡
핵심 가치: 마케팅과 서비스 상호작용을 동일한 채널(동일 번호)에서 처리하여 고객에게 일관된 경험을 제공합니다. Data 360이 모든 메시지의 기반이 됩니다.

지원 에디션 및 요구사항

이메일·SMS Unified Messaging
Salesforce Enterprise 및 Unlimited Edition (Service Cloud) AND Marketing Cloud Next Growth·Advanced Edition
WhatsApp Unified Messaging
Marketing Cloud Engagement Growth·Advanced Edition (Enterprise, Unlimited)
🚫
Government Cloud Plus에서는 지원되지 않습니다.

Unified Messaging 이해

WhatsApp Unified Messaging

Unified Conversation Platform을 기반으로 서비스와 마케팅 상호작용을 하나의 채널에서 제공합니다. 단방향 마케팅 프로모션과 양방향 고객 지원 간 전환이 용이하며, Data 360이 모든 메시지를 지원합니다.

  • 하나의 WhatsApp 번호로 마케팅+서비스 상호작용 모두 처리
  • 기존 WhatsApp 번호·WABA 재사용 가능 (마이그레이션 지원)
  • 고객의 전체 프로필(마케팅+지원 이력) 확인 가능

이메일·SMS Unified Messaging

이메일·SMS Unified Messaging은 Data 360과 마케팅 활동을 연결하는 프레임워크를 제공합니다. 구독자 인구통계 및 동의 정보를 Data 360에 저장하고, 세그먼트·타겟팅으로 의미 있는 상호작용을 구현합니다.

SMS Unified Messaging 라우팅 방식

상호작용 유형라우팅 처리
서비스 관련 인터랙션Omni-Channel → 봇 또는 서비스 에이전트 라우팅
마케팅 관련 인터랙션Flow → 마케팅 처리 방식 결정
🚫
기존 Service Cloud Messaging 채널이나 Marketing Cloud Engagement 채널을 Unified Messaging 채널로 업그레이드하는 것은 지원되지 않습니다. SMS Unified Messaging 설정 시 새 전화번호를 사용해야 합니다.

설정 가이드

Data 360 연결 (필수 첫 번째 단계)

메시지 발송 전 Data 360에 연결하고 필수 데이터 키트를 배포해야 합니다. 데이터 키트는 관리 패키지와 유사하며, 인게이지먼트 추적과 동의 처리에 필요한 데이터 스트림을 생성합니다. 채널별로 Data 360 연결을 생성해야 합니다.

📌
각 채널 유형에는 2개의 데이터 키트가 필요합니다. 하나는 동의 설정, 다른 하나는 채널 인게이지먼트 데이터용입니다. 동시 설치 또는 순차 설치 모두 가능합니다.
⚠️
올바른 데이터 처리를 위해 Data Cloud Salesforce Connector 권한 세트의 필드 가시성을 유지하세요. 필드 삭제, 권한 변경, API 이름 변경 시 가시성이 변경될 수 있습니다.

데이터 키트 설치 방법

1
Setup → Quick Find → Unified Messaging → 구성할 채널 선택
2
Channel Setup 섹션 → Install 클릭
  • 이메일: Marketing Cloud Next 설정 프로세스의 일부로 설치
  • SMS: Unified Messaging SMS 섹션의 SMS Codes에서 데이터 키트 찾기
  • WhatsApp: Unified Messaging WhatsApp 섹션의 Your Numbers에서 데이터 키트 찾기
3
설치 완료 후 Done — Data 360에 필수 데이터 스트림 연결 완료

브랜드 도메인 생성

브랜드 도메인을 사용해 URL을 개인화된 단축 링크로 변환하고 인터랙션을 추적합니다. 유효한 CA 서명 HTTPS 인증서가 필요합니다. 단축 링크는 90일 후 만료됩니다.

1
Setup → Quick Find에서 도메인 페이지 검색
  • SMS 도메인: URL Shortening Domain
  • 이메일 도메인: Links (Unified Messaging 이메일 섹션)
2
New Domain → 도메인 이름 입력 → CA 서명 HTTPS 인증서 선택 (https:// 자동 추가)
3
도메인 저장 → 드롭다운 → Details
4
DNS CNAME 레코드 설정 (IT팀과 협업하여 도메인 레지스트라에서 DNS 업데이트)
5
DNS 정보 전파 후 Status 클릭하여 도메인 활성화

WhatsApp 채널 설정

Unified WhatsApp 설정은 Data Cloud, Marketing Cloud, Service Cloud, Meta 4곳에서 진행합니다.

📌
필요 패키지 버전: Salesforce Standard Data Model 패키지 1.69 이상, UnifiedMessagingConsent 패키지 1.3 이상. 이전 버전에서 업그레이드해도 기존 동의 데이터는 유지됩니다.

권장 설정 순서

1
Data 360 설정 — 마케팅·서비스 클라우드의 데이터를 통합
2
WhatsApp Business Account(WABA) 및 번호 연결 — Salesforce 계정에 연결 (WABA 없으면 새로 생성)
3
컴플라이언스 키워드 설정
4
크로스클라우드 관계 설정 — 사용하는 마케팅 시스템(MCE 또는 MCN) 및 Service Cloud 연동 여부 설정
5
Marketing Cloud 라우팅 설정 — 고객을 마케팅 캠페인에 유지할지, 고객 지원으로 보낼지 결정
6
Service Cloud 라우팅 설정 — 사용자·큐·Omni-Flow(에이전트/봇 조합) 중 라우팅 방식 선택

기본 클라우드 및 라우팅 설정

수신 고객 메시지를 MCE, MCN, Digital Engagement 중 어디로 연결할지 결정합니다.

기본 클라우드동작 방식권장 시나리오
Service Cloud (권장) 모든 수신 메시지를 서비스로 라우팅. 마케팅 캠페인 중 수신 메시지도 서비스로 연결 고객 지원 중심, 수신 메시지 대부분이 지원 문의인 경우
Marketing 모든 수신 메시지를 마케팅 Flow(MCN) 또는 Journey(MCE)로 라우팅. 오픈 Service Cloud 세션이 없는 경우 마케팅 캠페인 응답이 주를 이루는 경우

마케팅→서비스 대화 전환

고객이 마케팅 캠페인에서 사전 정의된 마케팅 키워드 이외의 응답을 보내면, 응답이 Service Cloud로 라우팅되어 지원을 받게 됩니다.

  • 10-Digit Long Code 요청

SMS 코드 활성화
4
URL 단축 도메인 설정 (브랜드 도메인 참조)
5
SMS 동의 설정
📌
SMS Unified Messaging은 텍스트 전용 메시지만 지원합니다.

WhatsApp 번호 마이그레이션

기존 WhatsApp Business Account(WABA) 또는 번호를 Unified WhatsApp으로 마이그레이션할 수 있습니다.

번호 유형Unified Messaging 지원 여부

MCE WhatsApp 채널에서 사용 중인 번호 또는 MCE 연결 WABA✅ 마이그레이션 가능
Service Cloud 강화된 WhatsApp 채널에서 활성화된 번호✅ 단, WABA가 여러 번호와 연결 불가
🚫
단일 WABA를 여러 Salesforce Org에 연결하는 것은 지원하지 않습니다. Sandbox와 Production은 별개의 Org로 취급됩니다.

동의(Consent) 관리

Unified Messaging의 각 채널은 별도의 동의 설정이 필요합니다. SMS Unified Messaging의 키워드와 응답을 커스터마이징할 수 있습니다.

  • SMS 동의: STOP/HELP 키워드 처리, 수신 거부 워크플로우 설정
  • WhatsApp 동의: 마케팅 vs 서비스 채널 동의 레벨 구분 설정

과금 고려사항

  • Unified Messaging 사용 시 메시지 발송당 과금이 발생합니다.
  • 단축 링크는 90일 후 만료됩니다. 만료된 링크는 구독자에게 "페이지를 찾을 수 없음" 오류를 표시합니다.
  • 비활성 Unified Messaging 채널은 자동으로 프로비저닝이 해제될 수 있습니다.
⚠️
Meta 규칙으로 인해 고객이 번호를 차단했거나, 30일 이상 오프라인이거나, 제재 국가에 있는 경우 메시지가 전달되지 않을 수 있습니다. 이 경우 Meta가 개인정보 정책상 미전달 사유를 공개하지 않을 수 있습니다.
