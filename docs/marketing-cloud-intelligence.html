Marketing Cloud Intelligence (MCI)

최종 업데이트: 2026.06.19 · 출처: Salesforce 공식 문서 (740페이지)

🧩 개요

Marketing Cloud Intelligence(MCI)는 마케팅·광고 플랫폼, 웹 분석, CRM, 이커머스 등 다양한 소스의 데이터를 연결·통합·시각화하여 캠페인 성과를 최적화하는 독립형 분석 플랫폼입니다.

MCI는 Salesforce MCE와 별개의 독립 플랫폼으로, 마케터가 여러 채널(Google Ads, Meta, Kakao 등)의 데이터를 한 곳에서 통합 분석할 수 있도록 설계되었습니다.

핵심 역량 4가지

역량설명MCI 도구
Connect (연결)다양한 소스 데이터를 API·파일로 수집TotalConnect, API Connectors, Connect & Mix 탭
Harmonize (통합)서로 다른 소스 데이터를 동일 기준으로 병합Harmonization Center, Parent-Child, Data Fusion
Visualize (시각화)통합 데이터를 대시보드·차트로 표현Collections, Pages, Widgets, Visualize 탭
Act (행동)인사이트 기반 최적화 의사결정 지원Activation Center, Goals, Einstein Insights, Analyze & Act 탭

🔧 계정 및 워크스페이스 설정

계정 구조

MCI 계정은 복수의 워크스페이스(Workspace)로 구성됩니다. 워크스페이스는 독립적인 데이터 스트림·대시보드·리포트를 보유합니다.

  • 계정 레벨 — 계정명·로고, 허용 워크스페이스 수, 통화, 비밀번호 정책
  • 워크스페이스 레벨 — 각자 독립된 데이터·대시보드·권한

워크스페이스 분리 기준 예시

  • 클라이언트 단위 (광고 에이전시)
  • 브랜드·사업부 단위
  • 채널 단위 (퍼포먼스 vs 브랜드)
  • 국가·지역 단위
  • 역할 기반 뷰 분리 (임원용 vs 실무자용)

사용자 역할

역할권한
Account Admin계정 전체 관리, 워크스페이스 생성·삭제, 사용자 초대
Workspace Admin해당 워크스페이스 설정·데이터 스트림·사용자 권한
Editor대시보드·위젯 편집, 데이터 업로드
Viewer대시보드·리포트 조회만 가능

🗂️ 데이터 모델

MCI는 다양한 소스의 데이터를 Dimension(차원)과 Measurement(측정값)라는 두 축으로 정규화합니다. 이 구조가 올바르게 유지되어야 데이터 집계 오류를 방지할 수 있습니다.

📐 Dimensions (차원)

데이터셋을 설명하는 비수치형 정성적 속성입니다. 계산할 수 없으며, 필터링·분류·그룹핑에 활용됩니다.

Dimension 계층 구조 (Hierarchy)

Dimension 간에는 계층 관계가 있으며, 이를 올바르게 설정해야 데이터가 정확하게 집계됩니다.

계층 수준Dimension 예시
최상위 (Account)광고주, 브랜드
캠페인Campaign ID, Campaign Name
광고 그룹Ad Group ID, Ad Group Name
광고 소재Ad ID, Creative Name, Format
키워드Keyword, Match Type
지면/채널Publisher, Channel, Placement

주요 표준 Dimension

  • Campaign Name, Campaign ID
  • Ad Name, Ad ID, Creative Format
  • Publisher (Google Ads, Meta, Kakao 등)
  • Channel (Search, Display, Social, Video)
  • Date, Week, Month, Quarter
  • Country, Region, City
  • Device Type (Mobile, Desktop, Tablet)

📏 Measurements (측정값)

집계·계산 가능한 수치형 데이터입니다. KPI 설정과 성과 분석의 기반이 됩니다.

주요 표준 Measurement

카테고리Measurement 예시
노출·도달Impressions, Reach, Frequency
클릭Clicks, CTR (계산값)
비용Spend, CPC, CPM (계산값)
전환Conversions, Conversion Rate, CPA
수익Revenue, ROAS, ROI
동영상Video Views, VTR, Watch Time
소셜Likes, Shares, Comments, Followers
이메일Opens, Open Rate, Unsubscribes

계산 필드 (Calculated Fields)

기존 Measurement를 조합해 커스텀 지표를 만들 수 있습니다. 예: ROAS = Revenue / Spend × 100

📥 데이터 업로드

MCI에 데이터를 가져오는 방법은 크게 두 가지입니다: TotalConnect(자동 연동)와 API Connectors(플랫폼 직접 연동).

🔌 TotalConnect

CSV·Excel 파일을 수동 또는 FTP/SFTP로 업로드하는 방식입니다. API를 지원하지 않는 레거시 시스템이나 내부 데이터를 MCI로 가져올 때 사용합니다.

TotalConnect 지원 파일 형식

  • CSV, TSV (탭 구분자)
  • Excel (.xlsx, .xls)
  • JSON
  • XML

업로드 방법

  • 수동 업로드 — MCI UI에서 직접 파일 업로드
  • SFTP 자동화 — 지정된 SFTP 폴더에 파일을 놓으면 자동 수집
  • 이메일 수신 — 특정 이메일 주소로 첨부 파일 발송 시 자동 처리
  • Google Sheets 연동 — 구글 스프레드시트 자동 가져오기

🔗 API 커넥터

주요 광고·분석 플랫폼과 API로 직접 연동하여 데이터를 자동으로 수집합니다. 한 번 설정하면 지정한 주기로 자동 업데이트됩니다.

🔬 Granular Data Center

수억 ~ 수십억 행 규모의 세분화된 원시 데이터(Raw Data)를 처리하기 위한 프리미엄 기능입니다.

일반 데이터 스트림과의 차이

항목일반 데이터 스트림Granular Data Center
과금 기준행(Row) 수저장 용량(TB)
데이터 규모수백만 행수억 ~ 수십억 행
데이터 유형집계 데이터원시 이벤트·로그 데이터
활용 사례캠페인 성과 KPI키워드 레벨, 이벤트 로그, 지오데이터

적합한 데이터 유형

  • 키워드 레벨 광고 성과 데이터
  • 이벤트 레벨 클릭·노출 로그
  • 세분화된 지오(geo) 위치 데이터
  • 앱 세션·이벤트 로그
ℹ️
Granular Data Center는 마켓플레이스 Add-on Features 탭에서 별도 구매하는 프리미엄 기능입니다.

📊 데이터 시각화

MCI의 시각화 도구는 Visualize 탭에서 모두 관리됩니다.

시각화 구성 요소

구성 요소설명
Widget단일 차트·테이블·숫자 표시 블록. 대시보드의 기본 단위.
Page복수의 Widget을 배치하는 대시보드 화면.
Collection복수의 Page를 묶은 보고서 묶음. 공유 및 내보내기 단위.

지원 차트 유형

  • KPI 타일 (단일 지표 표시)
  • 막대 차트 (Bar, Horizontal Bar, Stacked)
  • 꺾은선 차트 (Line, Area)
  • 파이·도넛 차트
  • 산점도 (Scatter Plot)
  • 피벗 테이블
  • 퍼널 차트
  • 지도 (Geo Map)

공유 및 내보내기

  • Collection을 PDF·Excel·CSV로 내보내기
  • 이메일 예약 발송 (Scheduled Sharing)
  • 대시보드 링크 공유 (뷰어 권한 부여 필요)
  • Slack 연동 자동 발송

🔍 데이터 분석

MCI의 Analyze & Act 탭에서 다양한 분석 도구를 사용할 수 있습니다.

주요 분석 도구

도구기능
Activation Center 데이터 기반으로 다음 마케팅 액션을 추천하는 실행 허브
Einstein Marketing Insights AI 기반 성과 이상 탐지·자동 인사이트 생성. 수동 분석 없이 개선 기회 발견.
Goals (목표 관리) KPI 목표값 설정 후 달성률을 실시간으로 모니터링
Pivot Tables 다차원 데이터를 피벗 형태로 자유롭게 슬라이싱·다이싱
Reports Dimension·Measurement 조합으로 커스텀 리포트 생성 및 예약 발송

🔀 데이터 하모나이제이션

서로 다른 플랫폼에서 수집한 데이터는 필드명·집계 방식·지표 정의가 제각각입니다. 하모나이제이션은 이를 통합 기준으로 정렬하여 전체 그림을 볼 수 있게 합니다.

하모나이제이션 도구

도구용도예시
Harmonization Center 소스별 필드를 공통 Dimension으로 매핑 Google의 "campaign_name"과 Meta의 "ad_set_name"을 "Campaign Name"으로 통합
Parent-Child 계층 관계 설정으로 상위 Dimension에 하위 집계 세 개 캠페인을 하나의 "Q2 브랜드 캠페인"으로 묶어 집계
Data Fusion 서로 다른 소스의 데이터를 공통 키로 병합 광고 지출(Meta) + 전환 데이터(GA) + 매출(CRM)을 캠페인 ID로 조인
💡
UTM 파라미터 일관성이 핵심Data Fusion을 통한 크로스 소스 분석이 잘 작동하려면 모든 캠페인에 일관된 UTM 파라미터 명명 규칙을 적용하는 것이 선행되어야 합니다. UTM이 불일치하면 조인 키를 찾지 못해 데이터가 누락됩니다.

💰 Media Transparency Center

미디어 예산 집행 계획(Media Plan)과 실제 집행(Delivery) 데이터를 비교하여 예산 페이싱, 비용 추적, 캠페인 최적화를 지원하는 프리미엄 모듈입니다.

주요 기능

  • Media Plan 업로드 — 예산 계획 데이터를 수동 또는 자동으로 업로드
  • Pacing 모니터링 — 예산 소진 속도가 계획 대비 빠른지/느린지 실시간 확인
  • 비용 추적 — 채널·캠페인·기간별 실제 집행 비용 vs 계획 비교
  • 이상 탐지 — 예산 초과 위험 알림, 미집행 예산 경고
ℹ️
Media Transparency Center는 Add-on Features 탭에서 별도 구매하는 프리미엄 기능입니다.

🏪 마켓플레이스 (Marketplace)

MCI 마켓플레이스에서는 추가 커넥터, 대시보드 템플릿, 분석 앱을 설치하여 기능을 확장할 수 있습니다.

마켓플레이스 카테고리

  • Add-on Connectors — 기본 제공 외 추가 API 커넥터
  • Dashboard Templates — 업종별·목적별 사전 구성 대시보드
  • Add-on Features — Granular Data Center, Media Transparency, Sandbox 등 프리미엄 기능
  • Partner Apps — 서드파티 분석 도구 연동
ℹ️
마켓플레이스는 누구나 탐색할 수 있지만, 설치·구매는 Workspace Admin 이상 권한이 필요합니다.

🎫 지원 케이스 등록

MCI 지원 케이스는 help.salesforce.com이 아닌 MCI 전용 채널을 통해 등록합니다.

  1. help.salesforce.com → Contact Support 클릭
  2. Trailblazer.account 로그인 (MCI 계정 이메일 사용)
  3. 대상 org 선택 → Get Support → Create a Case
  4. 다음 정보 입력:
    • Product: Intelligence
    • Topic: Administration/Settings/Marketplace · Analyze & Act · Connect & Mix · Visualize 중 선택
    • Workspace ID: MCI 화면 좌상단 워크스페이스 이름 클릭 시 확인 가능
    • 문제 발생 페이지 URL
