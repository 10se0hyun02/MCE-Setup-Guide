# 회의록 → 대시보드 추출 Agent 루틴

회의 후 Claude에게 회의록을 붙여넣으면 할일·이슈·커뮤니케이션 기록을 자동 추출합니다.
**등록 일정:** 수동 실행 (회의 직후)

---

## 사용 방법

1. Claude 대화 시작
2. 아래 프롬프트 + 회의록 내용을 함께 붙여넣기
3. Agent가 `import-queue.json` 생성
4. MCE 대시보드 → `🤖 브리핑` 탭 → **📥 회의록 가져오기** 버튼 클릭
5. 미리보기 확인 후 **가져오기** 클릭

---

## 루틴 프롬프트 (복사하여 사용)

```
아래 작업을 수행해주세요:

── 1. 회의록 읽기 ──────────────────────────────
먼저 파일을 읽어주세요:
- G:\내 드라이브\Projects\MCE-Setup-Guide\meeting-draft.txt

(파일이 없으면 "meeting-draft.txt 파일 없음. 대시보드 회의록 탭에서 [Agent 추출] 버튼을 먼저 클릭하세요."라고 알려주세요)

── 2. 회의록 분석 ──────────────────────────────
아래 회의록을 분석해서 대시보드에 입력할 항목들을 추출해주세요.

분석 기준:
- 할일(todos): "~하기로", "~확인 필요", "~전달 예정" 등 액션 아이템
- 이슈: 문제점, 리스크, 결정 보류 사항
- 커뮤니케이션: 이 회의 자체의 기록

날짜 처리:
- 명시된 날짜가 없는 할일은 date를 회의 날짜로 설정
- "다음 주까지", "이번 주 내" 등 상대 표현은 절대 날짜(YYYY-MM-DD)로 변환

우선순위 기준 (이슈):
- high: 일정·예산·고객 영향 있는 것
- medium: 해결 필요하지만 급하지 않은 것
- low: 개선 사항, 참고 사항

결과를 아래 경로에 저장해주세요:
G:\내 드라이브\Projects\MCE-Setup-Guide\import-queue.json

저장 완료 후 meeting-draft.txt 파일을 삭제해주세요:
G:\내 드라이브\Projects\MCE-Setup-Guide\meeting-draft.txt

포맷:
{
  "source": "회의록",
  "meetingDate": "YYYY-MM-DD",
  "meetingTitle": "회의 제목",
  "participants": "참석자1, 참석자2",
  "processedAt": "ISO 날짜시각",
  "todos": [
    {
      "content": "할일 내용",
      "date": "YYYY-MM-DD",
      "assignee": "담당자 (없으면 빈 문자열)",
      "memo": "추가 맥락 (없으면 빈 문자열)"
    }
  ],
  "issues": [
    {
      "title": "이슈 제목",
      "priority": "high | medium | low",
      "description": "상세 내용",
      "status": "open"
    }
  ],
  "communications": [
    {
      "date": "YYYY-MM-DD",
      "type": "대면미팅",
      "participants": "참석자1, 참석자2",
      "summary": "회의 주요 내용 3~5문장 요약",
      "nextAction": "다음 액션 (없으면 빈 문자열)"
    }
  ]
}

--- 회의록 시작 ---
[여기에 회의록 붙여넣기]
--- 회의록 끝 ---
```

## 출력 파일

| 파일 | 설명 |
|------|------|
| `import-queue.json` | 대시보드 가져오기 대기 파일 (가져오기 완료 후 자동 삭제) |

---

## 추출 항목 예시

**입력 회의록:**
> 킥오프 미팅 (2026-07-01, 참석: 홍길동 담당, 김철수 PM)
> - IP Warming 일정 협의 → 7/15 시작으로 확정
> - 고객사 도메인 설정 확인 필요 (김철수 담당, 7/5까지)
> - 기존 HTML 템플릿 파일 전달 요청

**추출 결과:**
- 할일: "고객사 도메인 설정 확인" (2026-07-05, 김철수)
- 할일: "HTML 템플릿 파일 전달" (2026-07-01, 담당자 미정)
- 커뮤니케이션: 킥오프 미팅 기록 자동 추가
